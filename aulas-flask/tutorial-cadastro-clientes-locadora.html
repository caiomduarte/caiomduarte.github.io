PASSO 2

Configurando SQLite

Agora vamos criar a conexão com o banco SQLite. 


import sqlite3
from flask import g

DATABASE = "database/bdlocadora.db"

def get_db():

    if "db" not in g:

        g.db = sqlite3.connect(DATABASE)

        g.db.row_factory = sqlite3.Row

    return g.db

def close_db(e=None):

    db = g.pop("db", None)

    if db is not None:
        db.close()

PASSO 4

Criando o Cliente DAO 


def inserir_cliente(dados):

    db = get_db()

    db.execute("""
        INSERT INTO clientes
        (nome, email)

        VALUES (?, ?)
    """, (

        dados.get("nome"),
        dados.get("email")

    ))

    db.commit()

PASSO 7

Entendendo o BuildError 


BuildError:
Could not build url for endpoint

🔥 Esse erro acontece quando:

  • o nome da rota está incorreto
  • o url_for() aponta para uma função inexistente
  • a rota ainda não foi criada

📚 Etapas

1. Função Listar Clientes 2. Criando a Rota 3. Criando Template 4. Criando Tabela 5. Utilizando Jinja2 6. Testando Sistema

PASSO 1

Criando a Função listar_clientes()

Abra o arquivo cliente_dao.py e crie a função responsável por buscar todos os clientes cadastrados. 


from database.db import get_db

def listar_clientes():

    db = get_db()

    clientes = db.execute("""
        SELECT * FROM clientes
    """).fetchall()

    return clientes

PASSO 2

Criando a Rota Flask 


from database.cliente_dao import listar_clientes

@app.route("/clientes")
def clientes():

    clientes = listar_clientes()

    return render_template(
        "clientes/listar-clientes.html",
        clientes=clientes
    )

PASSO 3

Criando o Template HTML

Agora crie o arquivo: 


templates/clientes/listar-clientes.html

PASSO 5

Utilizando Jinja2

Agora vamos renderizar os dados dinamicamente usando Jinja2. 


{% for cliente in clientes %}



    {{ cliente.id }}
    {{ cliente.nome }}
    {{ cliente.rg }}
    {{ cliente.email }}
    {{ cliente.telefone }}



{% else %}



    
        Nenhum cliente encontrado.
    



{% endfor %}

💡 Resumindo:

  • {{ ... }} → exibe valores
  • {% ... %} → controla lógica

PASSO 6

Testando o Sistema 


python app.py

1

O que vamos construir?

Vamos criar um app de adivinhação onde o celular pensa em um número de 0 a 10 e o usuário tenta descobrir qual é. O app vai dar dicas (maior ou menor) a cada tentativa e contar quantas tentativas foram necessárias.

🎯
Objetivo Pedagógico

Este projeto usa os 5 componentes básicos do React Native de forma prática e integrada. Ao final, você vai entender o papel de cada um.

Funcionalidades do app

  • Gera um número aleatório de 0 a 10 ao iniciar
  • Mostra dicas: "tente um número MAIOR" ou "MENOR"
  • Conta o número de tentativas
  • Exibe mensagem de vitória ao acertar
  • Botão para jogar novamente
2

Mapa dos Componentes

Antes de codar, veja o papel de cada componente que usaremos:

ComponenteO que fazEquivalente Web
ViewContainer / caixa que agrupa outros componentes<div>
TextExibe textos na tela<p> ou <span>
TextInputCampo de entrada de texto/número<input>
TouchableOpacityBotão clicável com efeito de opacidade<button>
SafeAreaViewRespeita notch e barras do sistemaNão tem equivalente direto
StyleSheetOrganiza estilos de forma otimizadaCSS / className
💡
Dica

No React Native NÃO existe CSS nem HTML. Você estiliza tudo com JavaScript, usando o StyleSheet. Cada View é como uma caixa que organiza o layout.

Configurando o Ambiente

1

Criar o projeto

Abra o terminal, navegue até a pasta onde quer criar o projeto e execute:

terminal 🔒 somente leitura 
npx create-expo-app AdivinhaNumero --template blank
cd AdivinhaNumero
📦
O que é o --template blank?

Cria um projeto limpo, sem telas ou navegação pré-configuradas. É o ponto de partida ideal para aprender.

2

Instalar as bibliotecas para rodar na Web

O Expo precisa de dependências extras para executar no navegador:

terminal 🔒 somente leitura 
npx expo install react-dom react-native-web @expo/metro-runtime
🌐
Por que essas bibliotecas?

react-native-web traduz os componentes para HTML/CSS. O @expo/metro-runtime é o bundler que serve o app no browser.

3

Substituir o App.js

Abra o arquivo App.js na pasta do projeto e apague TODO o conteúdo. Agora vamos escrever do zero!

4

Construindo Passo a Passo

Parte 1 — Imports

Todo arquivo React Native começa importando o que vai usar. Escreva no App.js:

App.js 🔒 somente leitura 
import React, { useState } from 'react';
import {
  View,
  Text,
  TouchableOpacity,
  TextInput,
  StyleSheet,
  SafeAreaView,
  Alert,
} from 'react-native';
📌
Por que importar?

No React Native, NADA funciona sem importar primeiro. Cada componente precisa ser declarado na lista de importações.

Parte 2 — Componente e Estados

Logo abaixo dos imports, crie o componente principal e declare os estados com useState:

App.js 🔒 somente leitura 
export default function App() {

  // useState(valorInicial) → [valorAtual, setValor]
  const [numeroSecreto] = useState(Math.floor(Math.random() * 11));
  const [tentativa,  setTentativa]  = useState('');
  const [dica,       setDica]        = useState('');
  const [tentativas, setTentativas]  = useState(0);
  const [ganhou,     setGanhou]      = useState(false);

Cada estado guarda um valor diferente do jogo:

  • numeroSecreto — o número que o app sorteou
  • tentativa — o que o usuário está digitando
  • dica — mensagem exibida na tela (MAIOR, MENOR, Acertou!)
  • tentativas — contador de tentativas
  • ganhou — true/false para controlar o que aparece

Parte 3 — Lógica do jogo

Ainda dentro do componente App, adicione a função de adivinhar:

App.js 🔒 somente leitura 
  function adivinhar() {
    const numero = parseInt(tentativa);

    // Valida se o usuário digitou um número válido
    if (isNaN(numero) || numero < 0 || numero > 10) {
      Alert.alert('Ops!', 'Digite um número entre 0 e 10.');
      return;
    }

    const novas = tentativas + 1;
    setTentativas(novas);
    setTentativa(''); // Limpa o campo

    if (numero === numeroSecreto) {
      setDica(`Você acertou em ${novas} tentativa(s)!`);
      setGanhou(true);
    } else if (numero < numeroSecreto) {
      setDica('Tente um número MAIOR!');
    } else {
      setDica('Tente um número MENOR!');
    }
  }

Parte 4 — O JSX (interface visual)

O return define o que aparece na tela. Adicione após a função adivinhar:

App.js 🔒 somente leitura 
  return (
    <SafeAreaView style={estilos.safeArea}>
      <View style={estilos.container}>

        {/* Título */}
        <Text style={estilos.titulo}>Adivinhe!</Text>
        <Text style={estilos.subtitulo}>Número de 0 a 10</Text>

        {/* Dica — só aparece se dica !== '' */}
        {dica !== '' && (
          <Text style={estilos.dica}>{dica}</Text>
        )}

        {/* Contador */}
        <Text>Tentativas: {tentativas}</Text>

        {/* Campo de texto */}
        {!ganhou && (
          <TextInput
            style={estilos.input}
            placeholder='Digite seu palpite...'
            keyboardType='number-pad'
            value={tentativa}
            onChangeText={setTentativa}
          />
        )}

        {/* Botão */}
        <TouchableOpacity
          style={estilos.botao}
          onPress={ganhou ? reiniciar : adivinhar}
        >
          <Text style={estilos.botaoTexto}>
            {ganhou ? 'Jogar Novamente' : 'Adivinhar'}
          </Text>
        </TouchableOpacity>

      </View>
    </SafeAreaView>
  );
}

Código Completo

Se em algum momento você se perdeu, aqui está o App.js completo para conferir:

App.js — versão completa 🔒 somente leitura 
import React, { useState } from 'react';
import {
  View, Text, TouchableOpacity, TextInput,
  StyleSheet, SafeAreaView, Alert,
} from 'react-native';

export default function App() {
  const [numeroSecreto] = useState(Math.floor(Math.random() * 11));
  const [tentativa,  setTentativa]  = useState('');
  const [dica,       setDica]        = useState('');
  const [tentativas, setTentativas]  = useState(0);
  const [ganhou,     setGanhou]      = useState(false);

  function adivinhar() {
    const numero = parseInt(tentativa);
    if (isNaN(numero) || numero < 0 || numero > 10) {
      Alert.alert('Ops!', 'Digite um número entre 0 e 10.');
      return;
    }
    const novas = tentativas + 1;
    setTentativas(novas);
    setTentativa('');
    if (numero === numeroSecreto) {
      setDica(`Você acertou em ${novas} tentativa(s)!`);
      setGanhou(true);
    } else if (numero < numeroSecreto) {
      setDica('Tente um número MAIOR!');
    } else {
      setDica('Tente um número MENOR!');
    }
  }

  function reiniciar() {
    setTentativa(''); setDica('');
    setTentativas(0); setGanhou(false);
    Alert.alert('Novo Jogo', 'Recarregue o app para novo número!');
  }

  return (
    <SafeAreaView style={estilos.safeArea}>
      <View style={estilos.container}>
        <Text style={estilos.titulo}>Adivinhe!</Text>
        <Text style={estilos.subtitulo}>Número de 0 a 10</Text>
        {dica !== '' && <Text style={estilos.dica}>{dica}</Text>}
        <Text>Tentativas: {tentativas}</Text>
        {!ganhou && (
          <TextInput style={estilos.input}
            placeholder='Seu palpite...'
            keyboardType='number-pad'
            value={tentativa}
            onChangeText={setTentativa} />
        )}
        <TouchableOpacity style={estilos.botao}
          onPress={ganhou ? reiniciar : adivinhar}>
          <Text style={estilos.botaoTexto}>
            {ganhou ? 'Jogar Novamente' : 'Adivinhar'}
          </Text>
        </TouchableOpacity>
      </View>
    </SafeAreaView>
  );
}

const estilos = StyleSheet.create({
  safeArea:   { flex: 1, backgroundColor: '#0f172a' },
  container:  { flex: 1, alignItems: 'center',
                justifyContent: 'center', padding: 24 },
  titulo:     { fontSize: 40, fontWeight: 'bold', color: '#f8fafc' },
  subtitulo:  { fontSize: 16, color: '#94a3b8', marginTop: 8 },
  dica:       { fontSize: 18, color: '#38bdf8',
                marginTop: 16, fontWeight: 'bold' },
  input:      { width: '100%', height: 56, backgroundColor: '#1e293b',
                borderRadius: 12, color: '#f8fafc', fontSize: 22,
                textAlign: 'center', marginTop: 24 },
  botao:      { width: '100%', height: 54, backgroundColor: '#0ea5e9',
                borderRadius: 12, alignItems: 'center',
                justifyContent: 'center', marginTop: 16 },
  botaoTexto: { color: '#fff', fontSize: 18, fontWeight: 'bold' },
});

Desafios para Praticar

Depois que o app estiver funcionando, tente evoluir por conta própria:

Nível 1 — Iniciante

Mudar as cores

Altere as cores no StyleSheet. Tente usar cores diferentes para o botão de "Adivinhar" e "Jogar Novamente".

Nível 2 — Intermediário

Limite de tentativas

Adicione um limite de 5 tentativas. Se errar 5 vezes, exiba a resposta e um botão para recomeçar.

Nível 3 — Avançado

Histórico de palpites

Adicione um histórico das últimas tentativas usando um array no useState e a função .map() para renderizar.

Rodando o App

Com o App.js completo e salvo, é hora de ver o resultado! No terminal, dentro da pasta do projeto, execute:

terminal 🔒 somente leitura 
npx expo start

O terminal vai exibir um menu de opções. Pressione a tecla:

w

Abrir no navegador (Web)

O app vai abrir automaticamente no seu browser. É a forma mais rápida de testar sem precisar de celular ou emulador!

⚡
Dica importante

Sempre que salvar o App.js, o Expo atualiza o app automaticamente no navegador. Você não precisa reiniciar — é só salvar e ver o resultado!

Se quiser testar no celular, baixe o app Expo Go (App Store ou Google Play), conecte na mesma rede Wi-Fi do computador e escaneie o QR Code que aparece no terminal.
